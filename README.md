# Agropath

Agropath es un programa escrito en Python que busca estudiar y optimizar las rutas logisticas de producción de soya en Illinois, utilizando datasets y espacios reales para poder conseguir una herramienta utíl. El proyecto combina procesamiento de datos, entrenamiento de modelos y una interfaz simple para visualizar resultados.

## Requisitos
- Python 3.10+
- Consultar `requirements.txt`

## Elementos
- Interfaz TUI para visualizar mapas.
- Página web con los resultados y analisis.
