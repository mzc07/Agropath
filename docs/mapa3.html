Regresión lineal: Producción estimada de cosechas

Descripción

Este gráfico representa la regresión lineal aplicada a los datos de producción de soya con respecto al año en el estado de Illinois, EE.UU.

Análisis adicional

Se utilizo la libreria Scikit-learn para generar un modelo que estudie el dataset y a partir de este dato genere la regresión lineal adecuada.

Se eligio usar esta tecnica estadistica para poder estudiar como es que cambia anualmente la producción de soya y realizar una estimación del dinero ganado por la producción de soya.

Dataset

Utilizamos la libreria pandas para poder procesar un dataset de la pagina kaggle, que contiene información relevante sobre la producción de soya en Illinois.
