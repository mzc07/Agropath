Rutas de transporte

Información

En este mapa se muestran la ruta más optima que recorre la distancia entre las fincas de producción con el centro de acopio y el puerto.

La distancia promedio entre las fincas y el centro de acopio es de 245.27 kilometros, y la distancia del centro de acopio al puerto es de 211.85 km.

Folium utilizo OSMnx para crear un grafo a partir de los datos geográficos y uso el algoritmo de Dijkstra para poder encontrar la ruta más corta entre ambos puntos, de esta forma minimizando el coste de transporte.
